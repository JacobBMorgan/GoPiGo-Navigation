Features

A* algorithm implementation to identify shortest path.

Automated route navigation.

Modular design for easy extension

Controls

First map to traverse must be constructed.

-g = graph(width, height).

Walls must be be set manually
-g.walls = [(x,y) for x,y in (0,1,2,3,....)]

Running A* algorithms on g will return a correct "pathway" through the map.

-aStarSearch(g, start, goal)
start, goal =(x,y)

"translator" translates pathway from A* into a simple assembly-like launguage to instruct GoPiGo's movement.

"driver" function converts translator directions into GoPiGo movements./

Bugs & Problems

For Python we want to make sure the spacing is correct.

The GoPiGo needs trim adjusted frequently as it is difficult to have it move in straight lines.

Power consumption is high and batteries will need replaced often.

Built in GoPiGo functions for movement run indefinitely unless a sleep command interrupts them.
